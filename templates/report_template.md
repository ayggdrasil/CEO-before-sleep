### D. Methodology

This report was generated using CEO Before Sleep v3.0.0.
Architecture: 14 dimension workers run in parallel waves via the Agent tool. Within each worker,
6 expert roles (Reviewer, Problem Finder, Problem Solver, Researcher, Consultant, Competitor Shadow)
execute sequentially, each building on all prior roles' output. Wave 1 (Dims 1–10) runs simultaneously;
Wave 2 (Dims 11–14) follows, consuming wave1_summaries.md. A dedicated Synthesis Agent assembles the
final report and computes the Composite Investability Score. Phase 6 (Investor Memo) and Phase 7
(Competitor Radar) are optional extensions that run after the core analysis.
